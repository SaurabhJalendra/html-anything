A token is a small piece of text. It might be a word, part of a word, a space, or punctuation.
AI tools count tiny pieces.
First rule: tokens are how the model prepares text for its machinery.
Not one word at a time. Not one letter at a time. Usually something in between.

Page 7 - Pocket RecapNow it clicks
Three takeaways
  • A token is a small text chunk, not always a word.
  • Tokens become IDs so the model can process them.
  • Context windows, cost, and speed are all tied to token count.
One misconception
Counting words is only a rough estimate. Punctuation, spaces, and unusual words can change the token count.

Pocket Glossary

Token

A small chunk of text that the AI model processes. It can be a word, word part, space, punctuation mark, or symbol.

Token ID

The number assigned to a token in the model's vocabulary.

Context window

The maximum number of tokens the model can consider in one request.

Source Notes

This is a simplified explainer. Real tokenization differs by model and tokenizer, so exact counts vary.
